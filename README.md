# 📱 Blue Manager - Abydos Edition (v1.0.0)

> **"Uhe~ You've worked hard all day, Sensei. Let Oji-san keep an eye on your laptop battery for you!"**
> A lightweight Windows battery monitoring application inspired by Schale HQ and the Abydos Foreclosure Task Force, featuring Takanashi Hoshino! 🌸

## 🌟 Key Features
* **Visual Tracking:** Displays accurate, real-time battery status and percentage on your Windows PC.
* **Healing Presence:** Hoshino's voice lines change dynamically and adorably based on your current battery levels.
* **Lightweight & Portable:** No installation required! Just download and run (Portable app) without consuming system resources.
* **Open Source:** 100% safe and transparent. The source code is publicly available for everyone to learn and develop further.

---

## 📥 Download & Usage Guide

1. Look at the **right sidebar** of this GitHub repository and find the **Releases** section.
2. Click on the latest version and download the **`byphongapp.exe`** file.
3. Double-click the file to run the application instantly!

⚠️ **Note for first-time execution:**
Since the application is packaged using PyInstaller and does not have an expensive Microsoft digital certificate, Windows Defender might show a blue warning screen (**SmartScreen**).<img width="983" height="640" alt="BatteryTest Archive" src="https://github.com/user-attachments/assets/406d8c11-859a-40b1-a2e4-949415c20742" /><img width="983" height="640" alt="BatteryTest Archive" src="https://github.com/user-attachments/assets/5924db71-e6db-4eda-8b03-b1c959e3adbd" />
